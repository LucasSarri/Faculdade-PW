Qual é a diferença entre linguagem de programação Server Side e Client Side? Cite exemplos de cada tipo

A diferença entre uma linguagem de programação Server Side e Client Side é dada principalmente a onde ela será primariamente processada. Ou seja a Server Side será processada primariamente pelo servidor e será entregue ao usuário o resultado pronto, enquanto a Client Side é processada principalmente pelo computador cliente.

Exemplos de Linguagem Server Side

  • PHP
  • ASP
  • Perl
  • Java

Exemplos de Linguagem Client Side

  • HTML
  • CSS
  • JavaScript
  • AJAX
