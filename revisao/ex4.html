Web Semântica

O termo "Web Semântica" refere-se à visão do W3C da Web dos Dados Linkados.

É muito importante saber que a linkagem dos dados é possível com tecnologias como RDF, SPARQL, OWL, SKOS. O objetivo final da Web de dados é possibilitar com que computadores façam coisas mais úteis e com que o desenvolvimento de sistemas possa oferecer suporte a interações na rede.

Há um certo je ne sais quoi no ar quando se fala no tema, expressado principalmente pelo tecnocratas.
